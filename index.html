⚗️ สูตรผสมการ์ด (Fusion)

นำการ์ด 2 ใบในสำรับมาผสมกันได้ที่ 🏕️ จุดพักแรม

📖 วิธีเล่น

🎯 เป้าหมาย
ไต่หอคอยให้สูงที่สุด สู้มอนสเตอร์ทีละชั้น เก็บการ์ด/Relic ให้แกร่งขึ้นเรื่อยๆ

⚔️ การต่อสู้
เล่นการ์ดโดยใช้ 💎 มานา (รีเซ็ตทุกเทิร์น) — การ์ดโจมตี ป้องกัน เวทย์ บัฟ จบเทิร์นแล้วการ์ดในมือจะถูกทิ้ง แล้วจั่วใหม่

👹 อ่าน Intent ศัตรู
แถบบนบอกล่วงหน้าว่าศัตรูจะทำอะไร 4 เทิร์น — สีแดง 💀 = ตีตายถ้าไม่ตั้งรับ, สีม่วง 💥 = บอสกำลังชาร์จท่าไม้ตาย!

🔥 จุดอ่อนธาตุ
การ์ดที่ขอบเรืองแสงเขียว 💥 คือธาตุตรงจุดอ่อนศัตรู ดีลเพิ่ม 50%

⭐ Skill Tree
เก็บ Skill Point จากการสู้/event มาอัพสกิลเฉพาะสายของ Class (กดไอคอน ⭐ บน map)

💎 Relic
ของวิเศษติดตัวทั้ง run หาได้จาก Treasure/Event/บอส — เสริม build ข้าม Class ได้

🔮 Fusion
ผสมการ์ด 2 ใบตามสูตรเป็นการ์ดใหม่ที่แรงกว่า (ทำที่จุดพัก 🏕️)

📅 Daily Challenge
ทุกคนเจอ map เหมือนกันในแต่ละวัน แข่ง leaderboard กันแฟร์ๆ

💀 ระวัง!
ตายแล้วจบเลย (roguelike) — เข้าห้องสู้แล้วปิดเกมหนีไม่ได้ Save จะหาย
